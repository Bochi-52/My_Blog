# MarkDown语法



## 标题

**ctrl+123456**

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

### 三级标题

#### 四级标题

##### 五级标题

###### 六级标题



## 字体

```markdown
**hello world**
```

**hello world**

```markdown
*hello world*
```

*hello world*

```markdown
<u>hello world</u>
```

<u>hello world</u>

```markdown
~~hello world~~
```

~~hello world~~

```markdown
>>> 内容
```
